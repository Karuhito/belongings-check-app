# 持ち物チェックリストアプリ すっきりチェック

外出前の持ち物確認に使えるシンプルなチェックリストアプリです。学校・ジム・旅行など場面ごとにカテゴリを作り、持ち物を管理できます。

## 背景
### 持ち物チェックアプリ — 開発背景
#### きっかけ
このアプリを作ったのは、自分自身が「忘れ物」に悩んでいたからです。
学校に行くとき、友だちと出かけるとき、旅行のとき——出先で「あ、忘れた」と気づいた瞬間の後悔は、誰もが一度は経験したことがあるはず。自分もそんな経験を繰り返してきました。
メモアプリに書いておく、頭の中で確認する……いろいろ試したけれど、どれもしっくりこない。そんな「ちょうどいいアプリがない」という課題感から、このアプリの開発をスタートさせました。

#### 解決したかった課題
既存のToDoアプリや汎用メモアプリは、持ち物管理には少し使いにくいと感じていました。

- 毎回同じリストを一から書くのが面倒
- 「学校用」「旅行用」など、場面ごとに持ち物が違うのに、うまく整理できない
- シンプルに使いたいのに、機能が多くて迷う


#### このアプリのこだわり
シーン別にリストを切り替えられることが、このアプリ最大の特徴です。
「学校」「旅行」「部活」「病院」など、場面ごとにあらかじめリストを用意しておけば、次からはワンタップで切り替えるだけ。毎回ゼロから書く手間がなく、その日の状況に合わせてサッと確認できます。
また、シンプルで余計な機能を省いたUIにもこだわりました。忘れ物チェックという目的だけに集中できる、すっきりとした使い心地を目指しています。

---

## 機能

- **カテゴリ管理** — 場面ごとにカテゴリを作成・削除
- **タブ切り替え** — 「全て」タブで全カテゴリのアイテムを一覧表示、各カテゴリタブで絞り込み表示
- **アイテムの追加** — テキスト入力または Enter キーで追加（カテゴリタブのみ）
- **チェック切り替え** — トグルボタンで確認済み／未確認を切り替え
- **アイテムの削除** — カテゴリタブ表示時に各アイテムを個別削除
- **自動保存** — データはブラウザの localStorage に自動保存

## 使い方

### カテゴリを作る

1. タブバー右端の「＋」ボタンをクリック
2. カテゴリ名を入力して「追加」または Enter キー
3. 作成したカテゴリタブに自動で切り替わる

<img src="images/home.png">

### アイテムを追加する

1. カテゴリタブを選択（「全て」タブでは追加不可）
2. 入力欄にアイテム名を入力して「追加」または Enter キー

<img src="images/category.png">

### 持ち物を確認する

- トグルボタンをクリックで確認済み（緑）／未確認（灰）を切り替え
- 「全て」タブでは全カテゴリのアイテムをまとめて確認できる

<img src="images/toggle.png">

### カテゴリを削除する

1. 削除したいカテゴリタブをクリックしてアクティブにする
2. タブに表示される🗑アイコンをクリック（カテゴリとそのアイテムがまとめて削除される）
